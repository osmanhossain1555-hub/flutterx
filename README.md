# Finance Web App

A static finance dashboard built with:

- index.html
- style.css
- script.js

## Run locally

Open `index.html` in your browser.

## Deploy to GitHub Pages

1. Create a GitHub repository
2. Upload these files
3. In repository settings, enable GitHub Pages
4. Choose the `main` branch and root folder

## Features

- Dashboard layout similar to your screenshot
- Accounts and goals sidebar
- Budget cards and expense breakdown
- Transaction table with filters
- Add transaction modal
- Browser storage using localStorage
