# Finance Tracker PWA - Form Editors

This version includes:
- install button for supported browsers
- offline support after first load
- GitHub Pages ready
- form-based editors for all sections
- add/remove rows for accounts, goals, planner, budget categories, and transactions
- export/import JSON backup

## Publish on GitHub Pages
1. Create a GitHub repo
2. Upload all files
3. Enable Pages in repo settings
4. Open the Pages URL in Chrome or Edge on Android
5. Tap **Install App**

## Important
The install prompt only appears when:
- the app is served over HTTPS or GitHub Pages
- the browser supports PWA install
- the browser decides the app is installable

The button will open the install prompt when available, otherwise it shows guidance.
